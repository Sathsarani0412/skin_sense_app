# Skin Sense AI Skincare Analysis Application

## Overview

Skin Sense is an AI-powered mobile application developed using Flutter that helps users analyze facial skin conditions using deep learning. Users can upload facial images through the camera or gallery, receive AI-based skin condition predictions, monitor their skin health progress over time, and obtain personalized skincare recommendations.

## Features

### User Authentication

* User Registration (Sign Up)
* Secure Login
* Forgot Password Functionality

### Skin Image Analysis

Users can:

* Capture facial images using the device camera
* Upload images from the gallery
* Submit images for AI-powered skin analysis

### Skin Condition Detection

The AI model detects and analyzes:

* Dark Spots
* Wrinkles
* Blackheads
* Whiteheads
* Acne

### Prediction Results

The system provides:

* Percentage score for each detected skin condition
* Graphical representation of skin analysis results
* Identification of the most affected skin condition
* Personalized skincare recommendations

### Progress Tracking

Users can monitor skin health improvements over time by:

* Viewing previous analysis records
* Comparing current and past prediction percentages
* Tracking increases or decreases in skin condition severity
* Visualizing progress through charts and analytics

### Personalized Recommendations

Recommendations are generated based on:

* Dominant skin condition
* Prediction percentages
* Skin health trends

### Profile Management

Users can:

* View profile information
* Update personal details
* Manage account settings
